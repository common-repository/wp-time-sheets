Visualize your data and events with Time sheets.


== Description ==

Visual representation for your data/events in different time frames.

Just include plugin and configure your data. No external dependencies, Just a few steps to create simple sheets with beautiful HTML5 layout and some really delicious CSS to be customized by almighty you.

Plugin features -

1. light and dark theme
2. Customizable bar colors
3. Easy to use

Find out more details about plugin [here](https://geekymohan.wordpress.com/2017/03/20/time-sheets-with-wp/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-time-sheets` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Time Sheets screen to create sheets



= Plugin Configurations =

1. Goto WP admin > Time Sheets
2. Select `add new` menu under `Time Sheets`
2. Create beautiful sheet and publish
4. Copy Shortcode from right sidebar widget
5. Paste Shortcode in your page/post
6. Done !!!
